# 百度贴吧签到助手

一款轻量级 Chrome 浏览器插件，实现百度贴吧关注贴吧的自动签到功能。无需每天手动打开贴吧逐个签到，插件会在后台自动完成，让你不再遗漏任何一家贴吧的经验值。

## 功能特性

### 自动签到
- **后台定时签到**：插件每小时自动检查一次，发现未签到的贴吧自动完成签到，无需打开浏览器
- **点击即签**：点击浏览器工具栏的插件图标，打开弹窗后自动开始签到所有未签到的贴吧
- **单吧签到**：支持对某个贴吧单独签到或重试

### 状态展示
- **实时进度**：签到过程中实时显示每个贴吧的签到状态（未签到 / 签到中 / 已签到 / 失败）
- **统计信息**：顶部状态栏显示"已签 X/Y"的签到进度
- **颜色区分**：
  - 灰色 — 未签到
  - 蓝色 — 签到中
  - 绿色 — 已签到
  - 红色 — 签到失败

### 数据持久化
- 签到结果保存在浏览器本地存储中，关闭弹窗后再打开仍可查看今日签到状态
- 跨天后自动重置，重新识别为"未签到"

### 系统通知
- 后台自动签到完成后，通过 Chrome 系统通知告知签到结果（成功/失败数量）

## 安装方法

### 环境要求
- **Google Chrome 浏览器**（版本 88 及以上，支持 Manifest V3）
- 已登录 [百度贴吧](https://tieba.baidu.com)（插件依赖浏览器中的登录 Cookie）

### 安装步骤

1. **下载插件文件**

   将整个插件文件夹下载到本地，确保文件夹内包含以下文件：
   ```
   ├── manifest.json
   ├── popup.html
   ├── popup.css
   ├── popup.js
   ├── background.js
   ├── content.js
   └── lib/
       └── tieba-api.js
   ```

2. **打开扩展程序管理页面**

   在 Chrome 地址栏输入：
   ```
   chrome://extensions/
   ```

3. **开启开发者模式**

   在扩展程序页面右上角，打开「开发者模式」开关。

4. **加载插件**

   点击页面左上角的「加载已解压的扩展程序」按钮，选择插件所在的文件夹（包含 `manifest.json` 的那个目录）。

5. **确认安装成功**

   安装成功后，浏览器工具栏会出现插件图标。如果没有看到，点击工具栏的拼图图标 🧩，找到「百度贴吧签到助手」，点击图钉固定到工具栏。

6. **登录百度贴吧**

   确保已在 Chrome 浏览器中登录百度账号。可以访问 [https://tieba.baidu.com](https://tieba.baidu.com) 确认登录状态。

## 使用方法

### 自动签到（推荐）
安装完成后无需任何操作，插件每小时自动在后台检查并签到未签到的贴吧。签到完成后会弹出系统通知。

### 手动签到
1. 点击浏览器工具栏的插件图标
2. 弹窗打开后自动加载关注贴吧列表并开始签到
3. 实时查看每个贴吧的签到进度
4. 签到失败的贴吧可点击右侧按钮单独重试

### 刷新数据
点击弹窗右上角的刷新按钮，可重新加载关注列表并清除今日签到记录，重新开始签到。

## 权限说明

| 权限 | 用途 |
|------|------|
| `storage` | 保存签到结果和配置信息到浏览器本地 |
| `alarms` | 设置每小时定时任务，触发自动签到 |
| `notifications` | 签到完成后发送系统通知 |
| `*://tieba.baidu.com/*` | 访问百度贴吧接口，获取关注列表和执行签到 |

插件**不会**访问任何百度贴吧以外的网站，**不会**收集或上传任何个人数据，所有数据仅保存在浏览器本地。

## 常见问题

### 签到失败，提示"无法获取 tbs 参数"
**原因**：未登录百度账号。  
**解决**：在 Chrome 中打开 [百度贴吧](https://tieba.baidu.com) 并登录，然后重新打开插件。

### 提示"无法获取关注贴吧列表"
**原因**：未登录或网络异常。  
**解决**：确认已登录百度贴吧，检查网络连接后重试。

### 插件图标不见了
**解决**：点击 Chrome 工具栏的拼图图标 🧩，在列表中找到「百度贴吧签到助手」，点击图钉图标固定显示。

### 后台自动签到没有执行
**原因**：Chrome 浏览器完全关闭后，Service Worker 会停止运行。  
**解决**：保持 Chrome 浏览器在后台运行（最小化即可，无需保持窗口打开）。Chrome 的定时任务会在浏览器启动后自动恢复。

### 签到接口返回错误码
百度可能对签到频率有限制，插件已设置每个贴吧间隔 1.5 秒的延迟。如果仍有大量失败，建议等待几分钟后点击刷新重试。

## 技术栈

- **Manifest V3** — Chrome 扩展最新规范
- **Chrome Extension APIs** — `storage`、`alarms`、`notifications`
- **原生 JavaScript** — 无第三方依赖，轻量纯净
- **Fetch API** — 所有网络请求均使用原生 fetch，自动携带 Cookie

```
├── popup.css              # 弹窗样式

└── lib/
    └── tieba-api.js       # 贴吧 API 封装（签到接口、关注列表）
```
